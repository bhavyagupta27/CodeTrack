# 🚀 CodeTrack

A responsive placement preparation dashboard built using **HTML, CSS, Bootstrap, and JavaScript**.

---

## ✨ Features

- 🔐 Login Authentication using LocalStorage
- 👤 Dynamic User Profile
- 🌞 Dynamic Greeting
- 📊 Placement Readiness Progress Bar
- 📈 Dashboard Statistics
- ✅ Daily Goals with LocalStorage
- 🎯 Goal Completion Counter
- 📱 Responsive Offcanvas Sidebar
- 🚪 Logout Functionality

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
